# devops-netology

Склонировал удаленный репозиторий локально

## Игнорируемые файлы

При помощи файла .gitignore, система контроля версий будет игнорировать файлы, которые мы не хотим сохранять и 
отправлять на удаленный репозиторий.
Файлы которые будут проигнорированы:
- Локальные каталоги terraform (./terraform)
- Файлы .tfstate
- Журналы сбоев (crash.log)
- Все файлы которые могут содержать конфиденциальные данные, например пароли.
- Временные файлы
